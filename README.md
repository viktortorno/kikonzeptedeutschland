# KI-Konzepte Deutschland – Website

Statische Website (reines HTML/CSS/JS) für KI-Konzepte Deutschland.

## Seiten
- `index.html` – Startseite
- `ki-konzepte.html` – KI-Konzepte
- `reifegrad-analyse.html` – Reifegrad-Analyse

## Assets
- `assets/` – Logos (VT)
- `logos/` – Kundenlogos

## Veröffentlichung
Die Seite wird über **GitHub Pages** ausgeliefert (Branch `main`, Root-Verzeichnis).
Eine benutzerdefinierte Domain wird über IONOS (DNS) auf GitHub Pages verbunden.
